# Byte App

## Instalación APK

1. Descargar el APK del link proporcionado mediante el instalador de paquetes.
2. Dar permisos para confiar en la fuente de google drive.
3. Presionar instalar en modal de instalación.
4. Presionar opción para instalar sin analizar.

## Instalación Respositorio Android Studio
1. Descargar Android Studio y seguir los pasos de instalación. Recomendado instalar el emulador de android al momento de instalar.
2. Clonar el repositorio a través del link
```
https://github.com/jorgeLanza2000/ByteExam.git
```
3. Seleccionar la opción "Confiar en este proyecto"
4. En la barra lateral derecha seleccionar el administrador de dispositivos
5. En caso no exista ningún dispositivo se debe crear una a través del botón +
  - Buscar un dispositivo con el que se desea probar
  - Seleccionar el disposivio deseado y presionar siguiente
  - Presionar botón finalizar
6. Presionar el botón play en el dispositivo que se quiere emular
7. Presionar botón play en la barra de navegación en la parte superior de android studio
